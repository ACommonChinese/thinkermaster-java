JDBC

JDBC

  • 概念: Java DataBase Connectivity
  • 本质: 官方定义的一套操作关系型数据库的规则, 即接口. 各个数据库厂商实现这套接口, 提供数据库驱动jar包, 我们可以使用这套接口(JDBC)编程, 真正执行的代码是驱动jar包中的实现类
